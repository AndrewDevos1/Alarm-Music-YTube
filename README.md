# Alarm-Music-YTube

# Politica de Privacidade - Alarm Music YTube

Ultima atualizacao: 24 de junho de 2026

Alarm Music YTube e uma extensao de despertador para Chrome e Brave que permite criar alarmes usando links de midia informados pelo usuario.

Esta extensao nao e afiliada, associada, autorizada, endossada ou oficialmente conectada ao YouTube, Google, Spotify ou SoundCloud. Os nomes de servicos de terceiros sao usados apenas para indicar compatibilidade com links fornecidos pelo proprio usuario.

## Dados armazenados

A extensao pode armazenar localmente no navegador:

- alarmes criados pelo usuario;
- horarios dos alarmes;
- links de midia informados pelo usuario;
- historico de alarmes;
- preferencias da extensao.

## Armazenamento local

Os dados sao armazenados localmente no navegador por meio das APIs de armazenamento da extensao, como `chrome.storage`.

Na versao atual, a extensao nao envia alarmes, historico, links de midia ou preferencias para servidores proprios.

## Compartilhamento de dados

A extensao nao vende dados pessoais.

A extensao nao compartilha dados pessoais com terceiros.

Links fornecidos pelo usuario podem ser abertos em servicos externos, como plataformas de midia, de acordo com a acao escolhida pelo proprio usuario. Esses servicos externos possuem suas proprias politicas de privacidade.

## Permissoes

A extensao solicita permissoes apenas para:

- salvar dados localmente;
- agendar alarmes;
- mostrar notificacoes;
- abrir paginas necessarias para o funcionamento do alarme;
- abrir links de midia configurados pelo usuario.

## Servicos de terceiros

A extensao pode abrir links de midia fornecidos pelo usuario em servicos externos.

Esses servicos possuem suas proprias politicas de privacidade e termos de uso.

## Limitacoes

A extensao depende do navegador e do computador estarem ativos. Se o computador estiver suspenso, desligado ou sem o navegador em execucao, o alarme pode nao tocar no horario exato.

## Contato

Para duvidas sobre privacidade, use o e-mail de suporte informado na pagina da extensao na Chrome Web Store.
